List Element

The < li> tag defines a list item. The < li> tag is used in ordered lists(< ol>), unordered lists (< ul>), and in menu lists (< menu>). The < ol> tag defines an ordered list. An ordered list can be numerical or alphabetical.

Syntax

< ol> < li>content< /li> < li>over< /li> < li>here< /li> < /ol >

Example

< ol > < li>Hello< /li> < li>World< /li> < /ol>

Display

  1. Hello
  2. World

table Element

The < th> tag defines a header cell in an HTML table, and the < td> tag defines a row in an HTML table.

Syntax

< tr> < th>put< /th> < th>the < /th> < th> content here < /th> < /tr>

Example

< tr> < th>< th >< /th> < th>can < /th> < th> give you a title of a table < /th> < /tr> < tr> < td>< td > < /td> < td>can < /td> < td>give you the rows of a table< /td> < /tr>

Display

< th > can give you a title of a table
< td > can give you the rows of a table
